# vue-pdfjs2

Display PDFs in your Vue app as easily as if they were images.

## Quick start

Install [`vue-pdfjs2`][vue-pdf2-npm] via [npm](https://npmjs.com) or [yarn](https://classic.yarnpkg.com).

```sh
$ npm install vue-pdfjs2 [--save]
# or yarn
$ yarn add vue-pdfjs2
```
